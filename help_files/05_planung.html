Die Schichtplanung

Dies ist das Herzstück der Anwendung. Hier erstellen Sie den detaillierten Dienstplan für ein ausgewähltes Event.

Der typische Arbeitsablauf

  1. Wählen Sie oben das zu planende Event aus.
  2. Nutzen Sie "Aufgaben aus Vorlagen..." oder "Neue Schicht", um die Struktur zu erstellen.
  3. Markieren Sie eine Schicht und nutzen Sie "Helfer zuweisen". Der Dialog zeigt Ihnen verfügbare Helfer und warnt farblich (Rot), wenn ein Helfer bereits stark ausgelastet ist oder keine Pause hat.

Qualitätssicherung: Dienstplan prüfen

Mit dem Button "Dienstplan prüfen" können Sie jederzeit eine Validierung des aktuellen Plans durchführen. Das System prüft auf:

  • Unterbesetzung: Leere oder nicht voll besetzte Schichten.
  • Regelverstöße: Helfer, die trotz Einschränkung eingeteilt wurden.
  • Belastung: Helfer, die ohne Pause durcharbeiten oder mehr als 2 Schichten haben.
  • Führung: Schichten, denen ein qualifizierter Teamleiter fehlt.

Der Automatische Planungsvorschlag

Dieser Algorithmus füllt leere Plätze intelligent auf. Er beachtet dabei strenge Fairness-Regeln:

  1. Ressourcen-Schonung: Teamleiter werden primär für Führungsaufgaben eingesetzt und nicht als einfache Helfer "verschwendet", solange andere verfügbar sind.
  2. Gesundheitsschutz: Aufeinanderfolgende Schichten ohne Pause werden vermieden.
  3. Lastenverteilung: Es wird versucht, niemanden für mehr als 2 Schichten pro Event einzuteilen.
  4. Fairness: Mitglieder mit Malus-Punkten (wenig geleistete Arbeit) werden bevorzugt herangezogen.

Teamleiter und Kompetenzträger

  • Teamleiter werden mit dem Zusatz (TL) gekennzeichnet.
  • Mitglieder mit Kompetenz werden mit dem Zusatz (*) gekennzeichnet.

Der Export-Assistent

Über "Dienstplan exportieren" erstellen Sie PDF- oder Excel-Dateien.

Besonderheit beim PDF-Export: Wenn Sie im Modul "Events verwalten" PDF-Anhänge (z.B. Hygieneverordnung) hinterlegt haben, werden diese automatisch an das Ende des Dienstplans angefügt, sodass Sie eine einzige, vollständige Datei erhalten.
