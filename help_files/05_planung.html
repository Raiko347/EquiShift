Die Schichtplanung

Dies ist das Herzstück der Anwendung. Hier erstellen Sie den detaillierten Dienstplan für ein ausgewähltes Event.

Der typische Arbeitsablauf

  1. Wählen Sie oben das zu planende Event aus.
  2. Nutzen Sie "Aufgaben aus Vorlagen..." oder "Neue Schicht", um die Struktur zu erstellen.
  3. Markieren Sie eine Schicht und nutzen Sie "Helfer zuweisen".

Komfort-Funktionen beim Anlegen

  • Intelligente Vorbelegung: Wenn Sie eine neue Schicht anlegen, übernimmt das System automatisch das Datum und die Endzeit der vorherigen Schicht als neuen Startpunkt. Das erleichtert das Anlegen von aufeinanderfolgenden Schichten enorm.
  • Standard-Dauer: Die Standard-Länge einer neuen Schicht (z.B. 2 Stunden) können Sie in den globalen Einstellungen definieren.

Helfer zuweisen (Manuell)

Der Zuweisungs-Dialog bietet Ihnen mächtige Werkzeuge:

  • Sortierung: Über das Dropdown-Menü können Sie die Liste sortieren nach Eignung, Score (Fairness) oder Name.
  • Warnungen: Das System warnt farblich (Rot), wenn ein Helfer an diesem Tag bereits stark ausgelastet ist oder keine Pause hat.
  • Mehrfachauswahl: Sie können mehrere Personen gleichzeitig auswählen (mit gedrückter Strg- oder Shift-Taste) und gemeinsam zuweisen.

Qualitätssicherung: Dienstplan prüfen

Mit dem Button "Dienstplan prüfen" können Sie jederzeit eine Validierung des aktuellen Plans durchführen. Das System prüft auf:

  • Unterbesetzung: Leere oder nicht voll besetzte Schichten.
  • Regelverstöße: Helfer, die trotz Einschränkung eingeteilt wurden.
  • Jugendschutz: Helfer, die für "Bar" oder "Kasse" eingeteilt sind, aber das in den Einstellungen definierte Mindestalter noch nicht erreicht haben.
  • Belastung: Helfer, die ohne Pause durcharbeiten oder mehr als 2 Schichten an einem Tag haben.
  • Führung: Schichten, denen ein qualifizierter Teamleiter fehlt.
