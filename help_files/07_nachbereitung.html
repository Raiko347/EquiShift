Nachbereitung von Events

Die Nachbereitung ist ein entscheidender Schritt, um das Bonus/Malus-System mit Daten zu füttern. Hier wird festgehalten, wer seine Dienste tatsächlich geleistet hat.

Arbeitsablauf

  1. Setzen Sie im Modul "Events verwalten" den Status des Events auf "Abgeschlossen".
  2. Wechseln Sie zum Modul "Nachbereitung".
  3. Wählen Sie das abgeschlossene Event aus der Liste aus.
  4. Die Tabelle zeigt alle geplanten Dienste an. Der Standard-Status ist "Geplant".
  5. Zeitersparnis: Klicken Sie auf "Alle auf 'Erledigt' setzen", um den häufigsten Fall schnell zu erfassen.
  6. Passen Sie nun manuell die Ausnahmen an, indem Sie im Dropdown-Menü den korrekten finalen Status auswählen (z.B. "Nicht Erschienen").
  7. Wenn Sie "Erledigt (durch Vertreter)" auswählen, öffnet sich ein Dialog zur Auswahl des Ersatz-Helfers.
  8. Klicken Sie am Ende auf "Änderungen speichern". Erst dann werden die Punkte für das Ranking berechnet.
