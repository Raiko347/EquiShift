Events verwalten

In diesem Modul legen Sie Ihre Veranstaltungen an, verwalten deren Status und fügen wichtige Dokumente hinzu.

Funktionen

  • Neues Event anlegen: Öffnet einen Dialog, in dem Sie den Namen, den Zeitraum und den Status festlegen.
  • Event bearbeiten: Ermöglicht die Änderung der Daten sowie das Hinzufügen von Dateianhängen (siehe unten).
  • Event kopieren: Ein mächtiges Werkzeug, um ein Event (z.B. vom Vorjahr) zu duplizieren. Siehe Abschnitt "Events kopieren".
  • Event löschen: Entfernt ein Event und alle damit verbundenen Daten endgültig.

Dateianhänge (PDF)

Im Dialog "Event bearbeiten" finden Sie den Reiter "Anhänge". Hier können Sie PDF-Dateien (z.B. Hygieneverordnungen, Lagepläne, Sicherheitsunterweisungen) hinterlegen.

Wichtig: Diese Anhänge werden beim PDF-Export des Dienstplans automatisch an das Ende der Datei angefügt.

ACHTUNG: Prüfen Sie vor der Verteilung des Dienstplanes zwingend die Aktualität und Gültigkeit aller angehängten Dokumente! Veraltete Sicherheitsvorschriften können zu Haftungsrisiken führen.

Events kopieren

Wenn Sie auf "Event kopieren" klicken, öffnet sich ein Assistent. Das System berechnet automatisch das neue Enddatum basierend auf der Dauer des Ursprungs-Events.

Sie haben drei Kopier-Modi zur Auswahl:

  1. Nur Struktur: Kopiert nur das Event und die definierten Aufgaben (z.B. "Bar", "Kasse").
  2. Struktur & Schichten: Kopiert zusätzlich die Zeiteinteilung. Das Datum der Schichten wird automatisch auf den neuen Zeitraum verschoben.
  3. Komplett: Kopiert alles, inklusive der Helfer. Der Status der Zuweisungen wird auf "Geplant" zurückgesetzt.

Hinweis: Auch die Dateianhänge werden mitkopiert. Bitte prüfen Sie diese im neuen Event auf Aktualität!
