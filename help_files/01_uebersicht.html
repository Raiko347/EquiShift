Das Hauptkonzept: Dienst-Typ vs. Aufgabe

Um die Anwendung effektiv zu nutzen, ist es wichtig, den Unterschied zwischen einem Dienst-Typ und einer Aufgabe zu verstehen:

  • Ein Dienst-Typ ist eine allgemeine Kategorie von Arbeit (z.B. "Bar", "Aufbau"). Dienst-Typen sind global und werden im Menüpunkt "Dienst-Typen" definiert. An sie werden die Kompetenzen und Einschränkungen der Mitglieder geknüpft.
  • Eine Aufgabe ist die konkrete Umsetzung eines Dienst-Typs bei einem bestimmten Event (z.B. "Cocktailbar im Foyer" vom Typ "Bar"). Aufgaben werden in der "Schichtplanung" für ein Event erstellt.

Das Datei-Menü (oben)

Hier finden Sie wichtige Funktionen zur Verwaltung Ihrer Daten.

  • Neue Datenbank...: Erstellt eine komplett neue, leere Datenbankdatei. Ideal für den ersten Start.
  • Datenbank öffnen...: Ermöglicht den Wechsel zu einer anderen, bereits existierenden Datenbankdatei (z.B. von einem anderen Organisator).
  • Backup erstellen...: Erstellt eine Sicherheitskopie der aktuell geöffneten Datenbank an einem von Ihnen gewählten Ort.
  • Aktuellen Datenbank-Pfad anzeigen: Zeigt Ihnen, wo die aktuelle Datenbankdatei auf Ihrem Computer gespeichert ist.
  • Einstellungen...: Öffnet einen Dialog, in dem Sie die Schriftgröße, den Vereinsnamen und Details für den PDF-Export anpassen können.
